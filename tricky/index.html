Demo: Tricky CSS Facts

1. Tinggi Nav Tidak Muncul Jika Semua Elemen di Dalamnya Inline

Elemen inline seperti <a> tidak membuat tinggi kontainer terisi. Solusinya adalah atur display elemen menjadi inline-block atau block.

Link

2. Inline Elements Tidak Memiliki Tinggi dan Lebar

Tag seperti <a> tidak bisa diberi tinggi/lebar kecuali diubah menjadi inline-block.

3. Mengatasi Float dengan Overflow: Hidden

Kontainer yang isinya elemen float bisa menghilang. Tambahkan overflow: hidden atau gunakan clearfix untuk menampilkan tinggi kontainer.

4. Clearfix Hack

Selain overflow: hidden, gunakan ::after clearfix hack agar container membungkus float di dalamnya.

5. Margin Auto Tidak Akan Bekerja Tanpa Width

Untuk membuat elemen horizontal center dengan margin: auto, kamu harus menetapkan width.

Centered Box

6. Height 100% Tidak Akan Berfungsi Tanpa Height dari Parent

Jika kamu memberi height: 100%, pastikan parent-nya punya height yang ditentukan, misalnya 100vh atau fixed height.

7. Z-index Hanya Bekerja di Elemen Posisi

z-index hanya berlaku jika elemen memiliki posisi relative, absolute, atau fixed.

8. Display: Inline-block Menyebabkan Whitespace

Jika beberapa elemen inline-block sejajar, spasi antar tag HTML akan dihitung sebagai spasi visual.

9. Position Absolute Mengacu ke Parent yang Berposisi

Elemen absolute akan mengacu pada parent terdekat yang memiliki position: relative atau absolute.

10. Display: None vs Visibility: Hidden

display: none menghilangkan elemen sepenuhnya dari layout, sedangkan visibility: hidden hanya menyembunyikan visual, tapi tetap menyimpan ruangnya.

11. Margin: Auto Hanya Bekerja pada Elemen Block

Elemen inline tidak akan bisa center menggunakan margin: auto. Harus display: block atau inline-block dan memiliki width.

12. Position: Fixed Menempel ke Viewport

Elemen dengan position: fixed akan selalu menempel pada viewport, bukan pada parent-nya.

13. Satuan VH dan VW Bisa Terganggu oleh Address Bar di Mobile

Gunakan 100svh untuk memastikan tinggi 100% viewport sebenarnya di mobile modern.

14. Transform Tidak Memicu Reflow, Top/Left Memicu

transform lebih baik untuk animasi karena tidak menyebabkan layout ulang.

15. Tidak Semua Properti Bisa Dianimasikan

Misalnya display tidak bisa dianimasikan. Gunakan opacity atau transform sebagai gantinya.

initial akan mengembalikan ke nilai default CSS.
inherit akan mewarisi dari parent.
unset akan memilih antara initial/inherit tergantung sifat properti.

17. Posisi Teks Tengah Vertikal: Gunakan Line-Height, Bukan Height

Untuk memposisikan teks secara vertikal di tengah dalam elemen, gunakan line-height yang sama dengan tinggi elemen. Menggunakan height saja tidak akan menengahkan teks secara vertikal.
